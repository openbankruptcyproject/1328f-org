About This Directory

Case counts reflect bankruptcy filings across all chapters in the FJC Integrated Database research dataset (FY 2008–2024). Not all courts have equal representation in the dataset. Courts with higher case counts generally have more complete data coverage.

PACER links point to each court's CM/ECF system. A free PACER account is required for document access. Register at pacer.uscourts.gov.
